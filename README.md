#BIKE SHARING DASHBOARD 📊
---

## Setup Environment - Anaconda

```bash
conda create --name main-ds python=3.13.12
conda activate main-ds
pip install -r requirements.txt
```
---
## Setup Environment - Shell/Terminal

```bash
mkdir bike_project
cd bike_project
pipenv install
pipenv shell
pip install -r requirements.txt
```
---
## Run Streamlit App

```bash
streamlit run dashboard.py
```
